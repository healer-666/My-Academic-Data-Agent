# 中国居民消费价格指数（CPI）月度同比数据分析报告

## 数据概览

- **数据来源**：中国居民消费价格指数（CPI）月度同比数据（上年同月=100）
- **时间范围**：2025年6月至2025年10月（共5个月）
- **指标数量**：13个分类指标，涵盖总指数、食品烟酒、衣着、居住、生活用品及服务、交通通信、教育文化娱乐、医疗保健、其他用品及服务、非食品、消费品、服务、不包括食品和能源等核心分类。
- **数据规模**：原始数据13行×6列，清洗后保持相同结构，无缺失值。
- **数据解读**：所有数值均为同比指数，基准值100表示与上年同月价格持平，高于100表示同比上涨（通胀压力），低于100表示同比下降（通缩压力）。

## 方法说明

1. **数据清洗**：加载原始Excel文件，检查缺失值（无缺失），统一列名格式，确保“指标”列为字符串并去除空格，保存为CSV格式。
2. **数据转换**：将数据转置为时间序列格式（月份为行，指标为列），月份列转换为datetime类型以便按时间排序。
3. **描述性统计**：计算各指标在5个月内的均值、标准差、最小值、最大值等。
4. **趋势分析**：使用线性回归（`scipy.stats.linregress`）计算每个指标随时间（月份序号1~5）的斜率、p值和R²，识别统计显著趋势（p<0.05）。
5. **可视化**：
   - 多指标时间序列折线图
   - 标准化热图（按指标行标准化为Z-score）
   - 最新月份（2025年10月）横向条形图
   - 统计显著趋势斜率条形图
6. **领域知识检索**：通过搜索确认CPI指标定义、分类含义及正常阈值，确保解读符合宏观经济惯例。

## 核心假设检验结论

### 1. 统计显著趋势（p < 0.05）

在13个指标中，**7个指标**显示出统计显著的月度同比变化趋势：

| 指标 | 斜率（每月变化） | p值 | 趋势方向 |
|------|----------------|------|----------|
| 生活用品及服务类居民消费价格指数 | +0.34 | 0.042 | 上升（通胀） |
| 交通通信类居民消费价格指数 | +0.55 | 0.0004 | 强劲上升 |
| 医疗保健类居民消费价格指数 | +0.26 | 0.0015 | 上升 |
| 其他用品及服务类居民消费价格指数 | +1.13 | 0.042 | 大幅上升 |
| 非食品居民消费价格指数 | +0.20 | <0.0001 | 稳定上升 |
| 服务居民消费价格指数 | +0.07 | 0.035 | 温和上升 |
| 不包括食品和能源居民消费价格指数 | +0.12 | 0.0019 | 上升 |

**解读**：
- **交通通信类**上升最快（斜率0.55），可能反映能源价格、出行成本或通信服务价格上涨。
- **其他用品及服务类**上升幅度最大（斜率1.13），表明奢侈品、个人护理等服务价格快速上涨。
- **非食品**和**核心CPI（不包括食品和能源）**均呈现稳定上升趋势，反映基础通胀压力。
- **服务价格**温和上升，符合服务消费升级趋势。

### 2. 非显著趋势

- **总CPI**（居民消费价格指数）斜率接近零（-0.01，p=0.92），表明整体同比价格水平在5个月内基本稳定。
- **食品烟酒类**呈下降趋势（斜率-0.52）但未达统计显著（p=0.17），可能受季节性供应影响。
- **衣着类**、**居住类**、**消费品**等变化不显著。

### 3. 最新月份（2025年10月）价格状态

- **低于100（同比通缩压力）**：3个指标
  - 食品烟酒类（98.4）
  - 交通通信类（98.5）
  - 消费品（99.8）
- **高于100（同比通胀压力）**：10个指标
  - 其他用品及服务类（112.8）涨幅最大
  - 衣着类（101.7）、居住类（100.1）、生活用品及服务类（101.9）等均高于基准
- **等于100（价格稳定）**：无

## 结果解释

1. **整体通胀温和**：总CPI在99.6~100.2之间窄幅波动，最新值100.2略高于基准，显示整体通胀压力极低。
2. **结构性分化明显**：
   - **食品烟酒**持续低于100，反映农产品供应充足或需求疲软，对整体CPI形成下拉。
   - **交通通信**虽低于100但趋势快速上升，可能因能源价格回升、出行需求恢复。
   - **服务与非食品**稳步上涨，体现服务消费升级和核心通胀韧性。
3. **核心通胀（不包括食品和能源）**从100.7升至101.2，显示剔除波动项后通胀温和上行。
4. **其他用品及服务类**异常高企（112.8），可能包含金银珠宝、旅游服务、个人护理等高弹性消费品，反映消费升级或特定品类供需紧张。

## 讨论

### 宏观经济含义

- **低通胀环境**：整体CPI接近100，符合当前中国宏观经济低通胀特征，为货币政策提供空间。
- **食品通缩**：食品价格同比下跌有助于稳定民生支出，但可能影响农业收入。
- **服务通胀**：服务价格持续上升反映经济结构向服务消费转型，劳动力成本上升可能传导至服务价格。
- **核心通胀韧性**：核心CPI稳步上升提示内生通胀压力存在，需关注未来走势。

### 政策启示

1. **货币政策**：整体通胀低迷支持宽松取向，但核心通胀上升提示不宜过度宽松。
2. **产业政策**：交通通信价格快速上升可能需关注能源、物流成本调控；其他用品及服务类大涨或反映消费升级需求，可引导供给优化。
3. **民生保障**：食品价格下跌利好居民实际购买力，但需防范“谷贱伤农”。

### 数据局限

- **时间窗口短**：仅5个月数据，趋势外推需谨慎。
- **未加权平均**：分析使用未加权平均，实际CPI为加权指数，各分类权重不同。
- **缺少环比数据**：同比指数无法反映月度间价格变动动量，建议结合环比数据深入分析。
