# 无创产前检测（NIPT）数据集分析报告

## 数据概览
- **数据来源**：无创产前检测（NIPT）临床记录，包含孕妇基本信息、测序质量指标、染色体Z值及胎儿健康结局。
- **样本规模**：1070条记录，31个变量。
- **关键变量**：孕妇年龄、BMI、测序读段数、染色体13/18/21/X/Y的Z值、胎儿是否健康等。
- **数据完整性**：原始数据中部分日期字段（末次月经、检测日期）存在缺失（约37%），其余数值与分类变量已通过中位数/众数填补。

## 方法说明
1. **数据清洗**：读取Excel原始数据，统一列名格式，转换数值与日期类型，对缺失值采用列中位数（数值）或众数（分类）填补，删除完全重复行。清洗后数据保存为CSV。
2. **统计方法**：
   - 描述性统计（均值、标准差、分位数）。
   - Z值高风险判定：依据NIPT临床指南，|Z| ≥ 3 视为高风险（对应染色体非整倍体阳性）。
   - 假设检验：独立样本t检验（Welch）比较健康与不健康胎儿组的染色体Z值差异。
   - 相关性分析：测序质量指标间的Pearson相关系数热图。
   - 分组比较：IVF妊娠与自然受孕的年龄、BMI及胎儿健康比例。
3. **可视化**：采用出版级图表样式，支持中文字体，生成分布直方图、箱线图、小提琴图、热图、条形图等。

## 核心假设检验结论
1. **染色体Z值高风险比例**：
   - 13号染色体：2.15%（23例）|Z| ≥ 3。
   - 18号染色体：4.30%（46例）|Z| ≥ 3。
   - 21号染色体：0.47%（5例）|Z| ≥ 3。
2. **胎儿健康与Z值关联**：
   - 18号染色体Z值在健康与不健康组间差异显著（t = -2.655, p = 0.0114），不健康组的Z值更低（均值偏移）。
   - 21号染色体Z值差异亦显著（t = 2.265, p = 0.0290），不健康组的Z值更高。
   - 13号染色体Z值差异未达显著水平（p = 0.0508）。
3. **测序质量指标相关性**：GC含量与各染色体GC含量高度正相关（r > 0.8）；原始读段数与唯一比对读段数强相关（r ≈ 0.99）。
4. **IVF妊娠与胎儿健康**：IVF（试管婴儿）与IUI（人工授精）样本量较小（共17例），与自然受孕相比，胎儿不健康比例未见显著升高（条形图显示比例相近）。

## 结果解释
- **Z值分布**：绝大多数样本的Z值落在[-3, +3]的正常范围内，符合预期。18号染色体的高风险比例相对较高（4.30%），可能提示该染色体在检测中变异较大或存在技术偏差。
- **胎儿健康关联**：18号与21号染色体的Z值与胎儿健康状态显著相关，但效应方向相反。结合搜索获得的临床知识，Z值绝对值≥3通常提示染色体非整倍体高风险（如唐氏综合征T21、爱德华氏综合征T18）。本数据中不健康胎儿仅占3.55%，其中部分可能对应真正的染色体异常。
- **测序质量**：测序指标间相关性合理，表明数据质量可控。GC含量的一致性反映了测序文库制备的稳定性。
- **孕妇特征**：平均年龄28.9岁（范围21‑43），平均BMI 32.3，属于超重范围，可能与孕期体重增加有关。

## 讨论
- **临床意义**：Z值阈值|Z|≥3是国内外NIPT筛查的常用截断值。本数据集中高风险比例较低，尤其是21号染色体（0.47%），与文献报道的唐氏综合征人群发病率（约0.1‑0.2%）基本吻合。18号染色体高风险比例偏高需进一步核查是否为技术假阳性或人群特征所致。
- **局限性**：数据中“染色体的非整倍体”字段多数为“T18”（众数填补），可能掩盖真实结果；胎儿健康状态仅为“是/否”，缺乏具体诊断信息；IVF样本量过小，统计检验效力不足。
- **建议**：后续可结合超声、核型分析等金标准验证高风险样本的真实性；可探索机器学习模型整合Z值、孕妇年龄、测序质量等多指标提升筛查准确性。
