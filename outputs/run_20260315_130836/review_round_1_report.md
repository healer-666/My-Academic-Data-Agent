# 中国CPI分类同比指数分析报告 (2025年6月‑10月)

## 数据概览

- **数据来源**：本地Excel文件 `simple_data.xls`，包含13个CPI分类指标，时间跨度为2025年6月至10月，共5个月。
- **指标说明**：所有指标均为“上年同月=100”的同比指数，反映各分类价格相对于去年同期的变化。
- **样本规模**：13行（指标）× 5列（月份），总观测值65个。
- **数据质量**：原始数据无缺失值、无重复行，无需插补。
- **清洗后数据路径**：`outputs/run_20260315_130836/data/cleaned_data.csv`

## 方法说明

1. **数据清洗**：统一列名格式，确保“指标”列为字符串，检查极端值（未发现异常），保存为CSV。
2. **描述性统计**：计算各月份的平均值、标准差、极差、变异系数等。
3. **假设检验**：
   - 鉴于样本量小（N<30），优先采用非参数检验。
   - 对总体CPI（居民消费价格指数）进行单样本Wilcoxon符号秩检验，检验中位数是否等于100（中性通胀）。
   - 对食品烟酒类与衣着类CPI进行Mann‑Whitney U检验，比较两类价格指数的差异。
4. **效应量与置信区间**：
   - 报告秩二列相关系数（rank‑biserial correlation）作为效应量。
   - 使用自助法（bootstrap，5000次重复）计算中位数及中位数差值的95%置信区间。
5. **可视化**：生成时间序列图、热力图、平均CPI条形图、月度箱线图，均采用出版级样式并支持中文标签。

## 统计学治理说明

- **小样本警告**：数据仅含5个月，样本量极小，因此所有推断均基于非参数方法，避免对分布形态的强假设。
- **多重比较校正**：本次分析仅进行两次独立假设检验（总体CPI vs 100、食品 vs 衣着），未涉及多组比较，故未应用Bonferroni或Tukey HSD校正。
- **效应量与置信区间**：严格遵循APA规范，同时报告检验统计量、p值、效应量及95%置信区间。
- **因果声明限制**：本分析基于观测数据，未进行随机实验，因此仅描述关联与差异，不作出因果推断。

## 核心假设检验结论

### 1. 总体CPI是否偏离中性通胀（100）？
- **检验方法**：单样本Wilcoxon符号秩检验（中位数 = 100）。
- **结果**：统计量 = 196.000，p = 0.0825。
- **效应量**：秩二列相关系数 r = 0.311（小到中等效应）。
- **95% CI**：总体CPI中位数的自助法95%置信区间为 [99.800, 100.900]。
- **解释**：在α=0.05水平上，不能拒绝总体CPI中位数等于100的原假设（p > 0.05）。置信区间包含100，提示2025年6‑10月整体通胀水平与上年同期相比未出现统计显著的偏离。

### 2. 食品烟酒类与衣着类CPI是否存在差异？
- **检验方法**：Mann‑Whitney U检验（独立样本）。
- **结果**：U = 0.000，p = 0.0112。
- **效应量**：秩二列相关系数 r = 1.000（完全分离，极大效应）。
- **95% CI**：两类CPI中位数差值的自助法95%置信区间为 [‑4.300, ‑1.600]。
- **解释**：食品烟酒类CPI显著低于衣着类CPI（p < 0.05）。效应量极大，表明两类价格走势完全分离。中位数差值CI全为负，提示食品烟酒类CPI比衣着类低约1.6至4.3个百分点。

## 结果解释

### 描述性统计
- 各月份平均CPI在100.73至101.43之间波动，10月均值最高（101.43），6月最低（100.73）。
- 变异系数（CV）在2.46%至3.54%之间，显示各分类在月份间相对波动较小。
- “其他用品及服务类”CPI最高（108.0‑112.8），而“交通通信类”最低（96.3‑98.5），反映服务与非必需品类通胀压力高于交通通信。

### 时间趋势
- 多数分类CPI在5个月内保持平稳，波动幅度小于2个百分点。
- 食品烟酒类CPI呈现先降后升趋势（6月100.1 → 9月97.4 → 10月98.4），可能与季节性供应变化有关。
- 衣着类CPI稳定在101.6‑101.8，显示衣着价格持续温和上涨。

### 分类差异
- 平均CPI最高的三类为：其他用品及服务（109.5）、衣着（101.7）、生活用品及服务（101.6）。
- 平均CPI最低的三类为：交通通信（97.5）、食品烟酒（98.5）、消费品（99.5）。
- 食品烟酒类CPI持续低于总体CPI，可能反映食品供应充足或政策调控效果。

## 讨论

### 领域知识整合
根据检索的国家统计局资料，CPI同比指数以100为基准，高于100表示价格上涨，低于100表示价格下降。中国CPI涵盖8大类、268个基本分类，本数据集包含其中核心分类及衍生指标（如非食品、消费品、服务、不包括食品和能源等）。

### 经济含义
- **整体通胀温和**：总体CPI中位数接近100，且置信区间包含100，表明2025年6‑10月期间未出现显著的通胀或通缩压力，与近期中国低通胀环境相符。
- **结构性分化**：食品烟酒类CPI显著低于衣着类，反映消费价格内部存在明显结构性差异。食品价格相对稳定或下降，可能受益于农产品供应改善；衣着类价格持续高于100，可能与原材料成本、品牌溢价有关。
- **服务与非食品通胀**：“其他用品及服务类”CPI持续高企（>108），提示服务与非必需品类通胀压力较大，可能与人力成本上升、消费升级趋势相关。

### 方法局限
1. **样本量极小**：仅5个月数据，时间序列过短，无法捕捉季节性、长期趋势或周期波动。
2. **空间聚合**：数据为全国汇总，掩盖了城乡、区域间的异质性。
3. **指标覆盖**：仅包含部分分类，缺少环比指数、核心CPI（不包括食品和能源）的详细分解。
4. **非实验设计**：仅能描述关联，无法推断政策或经济事件对CPI的因果影响。

## 结论

2025年6月至10月，中国居民消费价格总体保持稳定，未出现统计显著的同比通胀偏离。然而，内部结构分化明显：食品烟酒类价格相对较低，衣着类价格温和上涨，其他用品及服务类价格涨幅较大。这种结构性差异提示通胀压力主要集中于非食品与服务领域，而食品价格对整体通胀形成下拉作用。

**政策建议**：监测重点应转向服务与非必需品类价格走势，同时关注食品价格波动对民生影响。未来研究需扩展时间跨度、纳入更多宏观经济变量，以深入理解通胀驱动因素。
