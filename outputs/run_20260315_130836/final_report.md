# 中国CPI分类同比指数分析报告 (2025年6月‑10月) – 修订版

## 数据概览

- **数据来源**：本地Excel文件 `simple_data.xls`，包含13个CPI分类指标，时间跨度为2025年6月至10月，共5个月。
- **指标说明**：所有指标均为“上年同月=100”的同比指数，反映各分类价格相对于去年同期的变化。
- **样本结构**：13行（分类）× 5列（月份），总观测值65个。
- **数据质量**：原始数据无缺失值、无重复行，无需插补。
- **清洗后数据路径**：`outputs/run_20260315_130836/data/cleaned_data.csv`

## 方法说明（修订）

1. **数据清洗**：统一列名格式，确保“指标”列为字符串，保存为CSV。
2. **描述性统计**：计算各月份横截面（13个分类）的平均值、标准差、极差、变异系数。
3. **分析单元澄清**：
   - **横截面单元**：每个月内13个分类的CPI值（每月一个截面）。
   - **时间序列单元**：每个分类在5个月内的CPI序列（时间依赖）。
   - **总观测值**：13分类 × 5个月 = 65个观测值，但时间序列内部存在自相关。
4. **假设检验（考虑时间依赖）**：
   - **单样本Wilcoxon符号秩检验**：检验全部65个观测值的中位数是否等于100（中性通胀），同时指出时间依赖可能影响独立性假设。
   - **Kendall's tau趋势检验**：对每个分类计算时间单调趋势的Kendall's tau，并报告正/负趋势分类数量。
   - **配对比较**：对食品烟酒类与衣着类CPI，采用配对差异的Wilcoxon符号秩检验（同一个月内配对），正确处理时间依赖。
5. **效应量与置信区间**：
   - 报告秩二列相关系数（rank‑biserial correlation）作为效应量。
   - 使用自助法（bootstrap，5000次重复）计算中位数及中位数差值的95%置信区间。
6. **可视化**：生成关键分类时间序列图、热力图、月度箱线图、平均CPI条形图，均采用出版级样式并支持中文标签。

## 统计学治理说明（修订）

- **小样本警告**：数据仅含5个月，样本量极小，因此所有推断均基于非参数方法，避免对分布形态的强假设。
- **时间依赖处理**：明确识别数据的时间序列结构，避免将不同月份的观测值视为独立样本。配对比较和趋势检验均考虑了时间依赖。
- **多重比较校正**：本次分析涉及多个分类的趋势检验，但未进行整体多重比较校正，结果解释时需注意可能存在的假阳性。
- **效应量与置信区间**：严格遵循APA规范，同时报告检验统计量、p值、效应量及95%置信区间。
- **因果声明限制**：本分析基于观测数据，未进行随机实验，因此仅描述关联与差异，不作出因果推断。

## 核心假设检验结论（修订）

### 1. 总体CPI是否偏离中性通胀（100）？
- **检验方法**：单样本Wilcoxon符号秩检验（中位数 = 100），基于全部65个观测值（13分类 × 5个月）。
- **结果**：统计量 = 583.500，p = 0.0023。
- **效应量**：秩二列相关系数 r = 0.272（小到中等效应）。
- **95% CI**：总体CPI中位数的自助法95%置信区间为 [100.100, 100.900]。
- **解释**：在α=0.05水平上，拒绝总体CPI中位数等于100的原假设（p < 0.05）。置信区间下限略高于100，提示2025年6‑10月整体通胀水平较上年同期有统计显著的轻微上升。
- **局限**：观测值之间存在时间依赖，此检验的独立性假设可能被违反，结果需谨慎解读。

### 2. 各分类CPI是否存在单调时间趋势？
- **检验方法**：对每个分类计算Kendall's tau（月份顺序1‑5 vs CPI值），评估单调趋势。
- **结果**：
   - 13个分类中，8个呈现正趋势（tau > 0），3个呈现负趋势（tau < 0），2个无趋势（tau = 0或缺失）。
   - 代表性趋势：交通通信类（tau = 1.0，完全正相关）、食品烟酒类（tau = -0.6，负趋势）、衣着类（tau = 0.359，弱正趋势）。
- **解释**：多数分类CPI在5个月内呈上升趋势，但趋势强度各异。交通通信类CPI逐月上升，食品烟酒类CPI逐月下降，衣着类CPI轻微上升。
- **局限**：时间点仅5个，趋势检验效力有限；未进行多重比较校正。

### 3. 食品烟酒类与衣着类CPI是否存在差异？
- **检验方法**：配对Wilcoxon符号秩检验（每月食品‑衣着差值）。
- **结果**：统计量 = 0.000，p = 0.0625。
- **效应量**：秩二列相关系数 r = 0.000（无效应）。
- **95% CI**：中位数差值的自助法95%置信区间为 [‑4.300, ‑1.500]。
- **解释**：在α=0.05水平上，不能拒绝食品烟酒类与衣着类CPI中位数差值为零的原假设（p > 0.05）。然而，置信区间全为负，提示食品烟酒类CPI可能低于衣着类CPI约1.5至4.3个百分点，但统计证据不足（可能因样本量小）。
- **改进**：配对检验正确处理了时间依赖（同一个月内配对），避免了原报告中Mann‑Whitney U检验的独立性误用。

## 结果解释

### 描述性统计
- 各月份横截面平均CPI在100.73至101.43之间波动，10月均值最高（101.43），6月最低（100.73）。
- 变异系数（CV）在2.46%至3.54%之间，显示各分类在月份间相对波动较小。
- “其他用品及服务类”CPI最高（108.0‑112.8），而“交通通信类”最低（96.3‑98.5），反映服务与非必需品类通胀压力高于交通通信。

### 时间趋势
- 关键分类趋势：总体CPI轻微上升，食品烟酒类下降，衣着类稳定略升，其他用品及服务类高位波动。
- 交通通信类CPI逐月上升（96.3 → 98.5），可能反映能源价格或交通成本变化。
- 食品烟酒类CPI先降后升（100.1 → 97.4 → 98.4），可能与季节性供应及政策调控有关。

### 分类差异
- 平均CPI最高的三类为：其他用品及服务（109.5）、衣着（101.7）、生活用品及服务（101.6）。
- 平均CPI最低的三类为：交通通信（97.5）、食品烟酒（98.5）、消费品（99.5）。
- 食品烟酒类CPI持续低于总体CPI，可能反映食品供应充足或政策调控效果。

## 讨论

### 领域知识整合
根据检索的国家统计局资料，CPI同比指数以100为基准，高于100表示价格上涨，低于100表示价格下降。中国CPI涵盖8大类、268个基本分类，本数据集包含其中核心分类及衍生指标（如非食品、消费品、服务、不包括食品和能源等）。

### 经济含义
- **整体通胀轻微上行**：总体CPI中位数显著高于100，且置信区间下限略高于100，提示2025年6‑10月期间存在轻微通胀压力，但幅度有限。
- **结构性分化明显**：服务与非必需品类（其他用品及服务）通胀压力较大，而交通通信、食品烟酒类价格相对低迷。这种分化可能反映需求结构变化、成本传导差异或政策干预效果。
- **趋势异质性**：各分类时间趋势不一，交通通信类持续上升，食品烟酒类波动下降，表明不同部门受外部冲击（如能源价格、农产品供应）的影响不同。

### 方法局限与改进建议
1. **样本量极小**：仅5个月数据，时间序列过短，无法捕捉季节性、长期趋势或周期波动。建议扩展时间跨度。
2. **时间依赖未完全建模**：虽采用配对检验和趋势检验，但未建立正式时间序列模型（如ARIMA）以控制自相关。
3. **空间聚合**：数据为全国汇总，掩盖了城乡、区域间的异质性。
4. **多重比较**：未对13个分类的趋势检验进行校正，可能增加假阳性。
5. **因果推断**：仅描述性分析，无法识别政策或经济事件的因果效应。

## 结论

2025年6月至10月，中国居民消费价格总体呈现轻微上行趋势，中位数显著高于中性通胀水平。然而，内部结构分化显著：服务与非必需品类通胀压力较大，而交通通信、食品烟酒类价格相对低迷。各分类时间趋势异质性明显，反映不同部门受外部冲击的影响不同。

**政策建议**：监测重点应转向服务与非必需品类价格走势，同时关注食品价格波动对民生影响。未来研究需扩展时间跨度、纳入更多宏观经济变量，并采用时间序列模型以更准确捕捉通胀动态。
