# 居民消费价格指数（CPI）分项数据分析报告

## 数据概览
- **数据来源**：中国居民消费价格指数（CPI）分项月度数据，基准为上年同月=100。
- **时间范围**：2025年6月至2025年10月，共5个月份。
- **指标数量**：13个分项指标，涵盖食品烟酒、衣着、居住、生活用品及服务、交通通信、教育文化娱乐、医疗保健、其他用品及服务、非食品、消费品、服务、不包括食品和能源等类别。
- **样本规模**：13行（指标）×5列（月份），总观测值65个。
- **数据质量**：原始数据无缺失值，无重复行，无需插补。

## 方法说明
1. **数据清洗**：读取Excel原始数据，检查缺失值与重复行，确保数值列格式正确，保存为CSV格式。
2. **描述性统计**：计算各月份所有指标的平均值、标准差、极差等。
3. **可视化**：
   - 时间序列图：展示各分项指标在5个月内的变化趋势。
   - 热力图：直观呈现各指标在不同月份的具体数值。
   - 月环比变化柱状图：展示总体CPI的月度变化。
4. **假设检验**：
   - 鉴于样本量极小（N=13），优先采用非参数检验。
   - 使用Friedman检验（非参数重复测量方差分析）判断各月份间是否存在显著差异。
   - 若Friedman检验显著，则进行配对Wilcoxon符号秩检验，并应用Bonferroni校正控制多重比较错误率。
   - 计算效应量Kendall's W，评估月份间差异的一致性程度。
5. **统计软件**：Python（pandas, numpy, scipy.stats, matplotlib, seaborn）。

## 统计学治理说明
- **小样本警告**：数据仅包含13个指标，每个指标仅5个时间点（N<30）。因此，所有推断统计均基于非参数方法，避免对正态分布的强假设。
- **多重比较校正**：若进行配对比较，采用Bonferroni校正，将显著性水平α调整为0.05/比较对数。
- **效应量与置信区间**：报告Friedman检验的效应量Kendall's W，并基于检验统计量提供定性解释。由于非参数检验的置信区间计算复杂，本报告未提供数值CI，但通过效应量补充了差异幅度信息。
- **因果声明限制**：本分析仅为观察性时间序列描述与关联分析，未建立因果机制。结论中避免使用“导致”“影响”等因果性词汇。

## 核心假设检验结论
1. **正态性检验（Shapiro-Wilk）**：总体CPI（居民消费价格指数）在5个月份的数值Shapiro-Wilk统计量=0.941，p=0.687（p>0.05），提示数据可能符合正态分布，但样本量极小，结论需谨慎。
2. **月份间差异检验（Friedman检验）**：
   - 检验统计量 χ² = 9.231，p = 0.056。
   - p值略高于0.05，表明在α=0.05水平上，各月份间差异未达到统计显著性。
   - 效应量Kendall's W = 0.192，提示月份间差异的一致性程度较低。
3. **多重比较**：由于Friedman检验未达到显著性阈值，未进行后续配对比较。

## 结果解释
- **描述性统计**：各月份指标均值在100.76至101.43之间波动，标准差在2.73至3.59之间，表明不同分项指数存在一定离散程度。极差最大为14.4（2025年10月），最小为11.1（2025年8月）。
- **时间序列趋势**：从时间序列图可见，多数分项指数在100附近波动，其中食品烟酒类指数在2025年9月降至最低（97.4），随后回升；衣着类指数相对稳定（约101.7）；交通通信类指数在2025年10月最高（112.8）。
- **月环比变化**：总体CPI在2025年6月至10月间变化较小，最大月环比下降0.5（2025年9月→2025年8月），最大月环比上升0.6（2025年8月→2025年7月）。
- **假设检验结果**：Friedman检验未发现各月份间存在统计显著差异（p=0.056），结合效应量较小（Kendall's W=0.192），提示在2025年6月至10月这五个月内，居民消费价格指数各分项的整体水平未发生系统性变化。

## 讨论
1. **经济含义**：CPI各分项在短期内保持相对稳定，可能反映该时期通胀压力较为温和。食品烟酒类指数先降后升，可能与季节性供需变化有关；交通通信类指数较高，可能受能源价格等因素驱动。
2. **方法局限性**：
   - 样本量极小（仅13个指标、5个时间点），统计检验力不足，可能掩盖实际存在的微小差异。
   - 时间跨度短（5个月），难以捕捉长期趋势或周期性波动。
   - 数据为同比指数（上年同月=100），无法直接反映环比价格变化。
3. **政策启示**：短期内CPI分项波动有限，但需关注交通通信等类别的高指数值，可能对居民生活成本构成压力。建议结合更长时间序列和环比数据进一步监测。
4. **未来研究方向**：可扩展时间范围至多年，引入环比指数，并利用面板数据模型控制个体（分项）固定效应，更精准识别时间趋势与结构性变化。
