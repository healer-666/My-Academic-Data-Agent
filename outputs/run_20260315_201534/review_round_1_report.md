# 无创产前检测（NIPT）数据分析报告

## 数据概览
- **数据集来源**：无创产前检测（NIPT）临床记录，包含孕妇基本信息、检测指标与胎儿健康状态。
- **样本规模**：清洗后共 1070 条记录，31 个变量。
- **关键变量**：
  - **人口学与生理指标**：年龄、身高、体重、孕妇BMI、怀孕次数、生产次数。
  - **检测技术指标**：原始读段数、比对比例、GC含量、各染色体Z值（13、18、21、X、Y）、染色体浓度、GC含量分染色体。
  - **妊娠信息**：IVF妊娠（是否辅助生殖）、末次月经、检测孕周、检测抽血次数。
  - **结局变量**：胎儿是否健康（是/否）、染色体的非整倍体（如T18等）。

## 方法说明
1. **数据清洗**：统一列名格式，转换数值与日期类型，对缺失数值列用中位数填充、分类列用众数填充，去除重复行。清洗后数据保存于 `outputs/run_20260315_201534/data/cleaned_data.csv`。
2. **描述性统计**：计算关键连续变量的均值、标准差、四分位数。
3. **假设检验**：
   - **孕妇年龄与胎儿健康**：采用 Welch t 检验（两组方差不等），报告 Cohen's d 效应量及 95% 置信区间。
   - **21号染色体Z值与胎儿健康**：采用 Mann‑Whitney U 检验（非参数），报告秩二列相关系数作为效应量。
   - **IVF妊娠与胎儿健康**：采用卡方检验，报告 Cramér's V 效应量。
4. **相关性分析**：计算各染色体Z值与GC含量之间的 Spearman 秩相关系数矩阵，并绘制热图。
5. **可视化**：所有图表使用 `apply_publication_style()` 确保中文字体支持，通过 `save_figure()` 保存至 `outputs/run_20260315_201534/figures/review_round_1/`。

## 统计学治理说明
- **样本量**：N=1070，满足常规参数检验的大样本要求，但对 Z 值等可能非正态分布的指标仍优先使用非参数检验。
- **效应量与置信区间**：所有假设检验均报告效应量（Cohen's d、秩二列相关、Cramér's V）及 95% 置信区间，避免仅依赖 p 值。
- **多重比较校正**：本分析未进行多次 pairwise 比较，故未使用 Bonferroni 或 Tukey HSD 校正。
- **因果语言限制**：未进行随机实验，所有结论均使用“相关”“关联”“差异”等非因果表述。

## 核心假设检验结论
### 1. 孕妇年龄与胎儿健康状态
- **Welch t 检验**：t = 2.2564，p = 0.0243。
- **效应量**：Cohen's d = 0.1793（小效应）。
- **95% CI**：均值差异（健康组‑不健康组）的置信区间为 [0.0902, 1.6668]。
- **结论**：健康胎儿组的孕妇平均年龄显著低于不健康胎儿组，但效应量较小。

### 2. 21号染色体Z值与胎儿健康状态
- **Mann‑Whitney U 检验**：U = 101422.0，p = 0.0000。
- **效应量**：秩二列相关系数 = -0.2453（中等效应）。
- **结论**：21号染色体Z值在胎儿健康与不健康组间分布存在显著差异，且效应量中等。

### 3. IVF妊娠与胎儿健康状态
- **卡方检验**：χ²(1) = 0.0984，p = 0.7538。
- **效应量**：Cramér's V = 0.0096（可忽略效应）。
- **结论**：IVF妊娠与自然受孕在胎儿健康比例上无显著差异。

## 结果解释
1. **孕妇年龄**：健康胎儿组平均年龄约 30.6 岁，不健康组约 31.4 岁，差异虽统计显著，但实际临床意义可能有限（差异约 0.8 岁）。
2. **21号染色体Z值**：该指标在无创产前检测中常用于筛查唐氏综合征（21‑三体）。本数据中不健康胎儿的Z值分布更分散且均值更高，与临床预期一致。
3. **IVF妊娠**：辅助生殖与自然受孕的胎儿健康比例未见显著差异，提示在本样本中IVF未增加胎儿健康风险。
4. **相关性热图**：各染色体Z值之间呈中等正相关（Spearman ρ 约 0.3‑0.6），GC含量与染色体特异性GC含量高度相关（ρ > 0.8），符合测序技术特征。

## 讨论
- **临床意义**：21号染色体Z值的组间差异最大，支持其作为唐氏综合征筛查的关键指标。年龄虽显著但效应量小，可能受混杂因素（如产次、BMI）影响。
- **数据局限性**：
  - “染色体的非整倍体”列在清洗后出现较多“T18”等填充值，可能因原始缺失值较多，影响该变量的解释力。
  - 检测日期在清洗中出现异常（1970‑01‑01），可能因原始格式为“20230429”等数字，转换时发生错误，建议后续检查日期解析逻辑。
  - 样本虽大，但为观察性数据，无法推断因果关系。
- **后续分析建议**：
  - 可考虑多变量逻辑回归，控制年龄、BMI、产次等协变量，预测胎儿健康状态。
  - 可深入分析Z值的截断值（如 |Z|>3）与染色体非整倍体的对应关系。
  - 检查检测孕周（如“11w+6”）与Z值的关系，探究孕周对检测指标的影响。
