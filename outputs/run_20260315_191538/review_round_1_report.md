# 中国居民消费价格指数（CPI）同比数据分析报告

## 数据概览

本数据集包含中国2025年6月至10月共5个月的居民消费价格指数（CPI）同比数据（上年同月=100）。数据涵盖13个指标类别，包括总指数、食品烟酒、衣着、居住、生活用品及服务、交通通信、教育文化娱乐、医疗保健、其他用品及服务等八大类及其细分项。数据规模为13行×6列，每个单元格为对应月份该类别的CPI同比指数。数据已清洗，无缺失值。

## 方法说明

1. **数据清洗**：读取原始Excel文件，确保列名规范，将指标列转为字符串，月份列转为数值型，保存为CSV格式。
2. **描述性统计**：计算各指标类别的均值、标准差、四分位数等。
3. **假设检验**：
   - 由于样本量小（N<30），优先采用非参数检验。
   - 使用Kruskal-Wallis H检验比较13个指标类别间的CPI是否存在显著差异。
   - 使用Friedman检验（重复测量非参数方法）检验5个月份间的CPI是否存在显著差异。
   - 若Friedman检验显著，则进行Wilcoxon符号秩配对比较，并应用Bonferroni校正控制多重比较错误率。
4. **效应量报告**：
   - Kruskal-Wallis检验报告ε²（epsilon squared）。
   - Friedman检验报告Kendall's W。
5. **可视化**：生成热图、趋势线图、箱线图和月度平均柱状图，均采用出版级美学风格，支持中文标签。

## 统计学治理说明

- **小样本警告**：总观测数65，每组仅5个观测，因此严格采用非参数检验，避免正态分布假设。
- **多重比较校正**：对月份间的10对比较采用Bonferroni校正，校正后α=0.005。
- **效应量与置信区间**：报告了ε²和Kendall's W，但因样本量小，未计算效应量的置信区间，在解释中注明此限制。
- **因果语言禁止**：分析仅揭示关联与差异，不声称因果关系。

## 核心假设检验结论

### 假设1：不同指标类别的CPI同比指数是否存在差异？
- **检验方法**：Kruskal-Wallis H检验（非参数）。
- **结果**：H = 58.94，p < 0.001。
- **效应量**：ε² = 0.9026（大效应）。
- **结论**：不同指标类别的CPI同比指数存在极显著差异（p < 0.001），效应量极大，表明类别间差异占总体变异的90.3%。

### 假设2：不同月份的CPI同比指数是否存在差异？
- **检验方法**：Friedman检验（重复测量非参数）。
- **结果**：χ² = 13.70，p = 0.0083。
- **效应量**：Kendall's W = 0.2634（中等效应）。
- **结论**：不同月份的CPI同比指数存在显著差异（p = 0.0083），效应量中等，表明月份间有一定一致性差异。

### 月份间配对比较（Bonferroni校正后）
- **校正方法**：Bonferroni校正，10次比较，校正后α=0.005。
- **显著配对**：仅“2025年8月 vs 2025年10月”达到显著（Wilcoxon W=3.00，p=0.0047）。
- **其他配对**：均未达到校正后显著性水平（所有p > 0.005）。

## 结果解释

1. **类别差异**：CPI同比指数在各类别间差异极大。其中“其他用品及服务类”指数最高（均值109.48，范围108.0–112.8），显著高于其他类别，反映该类别价格同比上涨较快。“交通通信类”指数最低（均值97.46，范围96.3–98.5），表明该类别价格同比有所下降。“居住类”指数完全稳定（均值100.10，无变异）。

2. **月份趋势**：从2025年6月到10月，整体CPI同比指数呈轻微上升趋势（月度均值从100.73升至101.43）。Friedman检验表明月份间存在显著差异，但配对比较仅发现8月与10月间差异显著（p=0.0047），其他相邻月份间差异不显著。这可能提示价格变化在8月后加速。

3. **领域知识整合**：根据检索结果，CPI同比指数以100为基准，高于100表示同比上涨，低于100表示同比下降。正常范围通常围绕100小幅波动（如101–103表示温和通胀）。本数据中多数类别接近100，总体通胀温和；但“其他用品及服务类”指数高达109–113，可能反映奢侈品、旅游服务等价格显著上涨；“交通通信类”指数低于100，可能与能源价格下降、通信资费下调有关。

## 讨论

- **小样本限制**：每个类别仅5个时间点，时间序列较短，限制了趋势分析的稳健性。效应量的置信区间未能精确估计，结论需谨慎外推。
- **经济含义**：CPI总体稳定，但内部结构分化明显。食品烟酒类指数波动较大（97.4–100.1），可能受季节性食品价格影响。居住类指数完全持平，反映房租及水电燃料价格高度稳定。
- **政策启示**：CPI结构分化提示货币政策需关注核心CPI（不包括食品和能源），本数据中“不包括食品和能源居民消费价格指数”均值为100.92，波动极小，显示核心通胀稳定。
- **关联非因果**：本分析仅描述价格指数的同期关联，不能推断月份变化导致类别差异，或类别差异引发总体变化。
