# 中国居民消费价格指数（CPI）同比数据分析报告（修订版）

## 数据概览

本数据集包含中国2025年6月至10月共5个月的居民消费价格指数（CPI）同比数据（上年同月=100）。数据涵盖13个指标类别，包括总指数、食品烟酒、衣着、居住、生活用品及服务、交通通信、教育文化娱乐、医疗保健、其他用品及服务等八大类及其细分项。数据规模为13行×6列，每个单元格为对应月份该类别的CPI同比指数。数据已清洗，无缺失值。

## 方法说明

1. **数据清洗**：读取原始Excel文件，确保列名规范，将指标列转为字符串，月份列转为数值型，保存为CSV格式。
2. **描述性统计**：计算各指标类别的均值、标准差、四分位数等，以及月度平均CPI。
3. **假设检验**：
   - 由于样本量小（N<30），优先采用非参数检验。
   - 使用Kruskal-Wallis H检验比较13个指标类别间的CPI是否存在显著差异（独立组设计）。
   - **纠正原错误**：原报告错误地将不同经济指标视为重复测量单元，误用Friedman检验。修订后，我们承认各月份数据在不同指标间存在配对结构（同一指标在5个月份上重复测量），但不同指标并非同一“区块”，因此不宜进行全局性的跨月份重复测量检验（如Friedman）。
   - 改为进行配对月份比较：针对连续月份（6月vs7月、7月vs8月、8月vs9月、9月vs10月）以及特别关注的8月vs10月，使用Wilcoxon符号秩检验（配对设计，以13个指标为配对单元），并报告效应量（匹配对秩二列相关系数r）。
   - 应用Bonferroni校正控制多重比较错误率。
4. **效应量报告**：
   - Kruskal-Wallis检验报告ε²（epsilon squared）。
   - Wilcoxon配对检验报告匹配对秩二列相关系数r（基于Z分数近似）。
5. **可视化**：生成热图、趋势线图、横向箱线图、带误差棒的月度平均柱状图以及10月与6月差异条形图，均采用出版级美学风格，支持中文标签。所有新图表保存至`review_round_2`目录。

## 统计学治理说明

- **小样本警告**：总观测数65，每组仅5个观测，因此严格采用非参数检验，避免正态分布假设。
- **多重比较校正**：对4对连续月份比较采用Bonferroni校正，校正后α=0.0125。对于额外关注的8月vs10月比较，若考虑所有10对可能比较，则校正后α=0.005。
- **效应量与置信区间**：报告了ε²和r，但因样本量小，未计算效应量的置信区间，在解释中注明此限制。
- **因果语言禁止**：分析仅揭示关联与差异，不声称因果关系。已删除原报告中“价格变化在8月后加速”等因果暗示表述。

## 核心假设检验结论

### 假设1：不同指标类别的CPI同比指数是否存在差异？
- **检验方法**：Kruskal-Wallis H检验（非参数，独立组）。
- **结果**：H = 58.94，p < 0.001。
- **效应量**：ε² = 0.9026（大效应）。
- **结论**：不同指标类别的CPI同比指数存在极显著差异（p < 0.001），效应量极大，表明类别间差异占总体变异的90.3%。

### 假设2：连续月份间CPI同比指数是否存在差异？（配对设计，以指标为配对单元）
- **检验方法**：Wilcoxon符号秩检验（配对非参数），Bonferroni校正。
- **结果**：
  - 6月 vs 7月：T=27.50，p=0.6212，r=0.1371（小效应），校正后不显著。
  - 7月 vs 8月：T=27.50，p=0.3653，r=0.2511（小效应），校正后不显著。
  - 8月 vs 9月：T=11.00，p=0.0494，r=0.5449（中等效应），校正后不显著（p > 0.0125）。
  - 9月 vs 10月：T=5.00，p=0.0126，r=0.6922（大效应），校正后不显著（p > 0.0125）。
- **额外关注比较**：8月 vs 10月：T=3.00，p=0.0047，r=0.7839（大效应）。若考虑所有10对月份比较的Bonferroni校正（α=0.005），此比较达到显著（p=0.0047 < 0.005）。
- **结论**：在严格控制多重比较后，连续月份间CPI差异均未达到统计显著水平。但8月与10月之间的差异在更严格的校正下显著，且效应量大，提示这两个月间可能存在有意义的变动。

## 结果解释

1. **类别差异**：CPI同比指数在各类别间差异极大。其中“其他用品及服务类”指数最高（均值109.48，范围108.0–112.8），显著高于其他类别，反映该类别价格同比上涨较快。“交通通信类”指数最低（均值97.46，范围96.3–98.5），表明该类别价格同比有所下降。“居住类”指数完全稳定（均值100.10，无变异）。

2. **月份趋势**：从2025年6月到10月，整体CPI同比指数呈轻微上升趋势（月度均值从100.73升至101.43）。配对检验显示连续月份间差异未达显著，但8月到10月的跨越比较显示显著差异（校正后p=0.0047），效应量大（r=0.78），提示在这两个月期间，多数指标类别的CPI发生了系统性上升。

3. **领域知识整合**：根据检索结果，CPI同比指数以100为基准，高于100表示同比上涨，低于100表示同比下降。正常范围通常围绕100小幅波动（如101–103表示温和通胀）。本数据中多数类别接近100，总体通胀温和；但“其他用品及服务类”指数高达109–113，可能反映奢侈品、旅游服务等价格显著上涨；“交通通信类”指数低于100，可能与能源价格下降、通信资费下调有关。

## 讨论

- **小样本限制**：每个类别仅5个时间点，时间序列较短，限制了趋势分析的稳健性。效应量的置信区间未能精确估计，结论需谨慎外推。
- **方法学纠正**：原报告错误使用Friedman检验，误将不同经济指标视为重复测量区块。修订后采用配对月份比较，更贴合数据结构（同一指标在不同月份配对）。
- **统计报告完整性**：所有配对比较均报告了检验统计量（Wilcoxon T）、p值和效应量（r），满足了APA报告标准。
- **经济含义**：CPI总体稳定，但内部结构分化明显。食品烟酒类指数波动较大（97.4–100.1），可能受季节性食品价格影响。居住类指数完全持平，反映房租及水电燃料价格高度稳定。
- **关联非因果**：本分析仅描述价格指数的同期关联，不能推断月份变化导致类别差异，或类别差异引发总体变化。已避免任何因果暗示表述。
