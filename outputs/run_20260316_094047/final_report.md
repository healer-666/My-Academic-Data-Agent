# 消费价格指数月度趋势分析报告 (2025年6月‑10月) – 修正版

## 数据概览

- **数据来源**：清洗后的数据集包含13项居民消费价格指数（CPI）及其细分类别，时间跨度为2025年6月至10月，共5个月份。
- **指标含义**：所有指数均为“上年同月=100”的同比指数，反映各月相对于上年同月的价格变化。
- **样本规模**：13项指标 × 5个月份 = 65个观测值。
- **数据质量**：原始数据无缺失值，已进行列名清洗、重复项检查，并保存为UTF‑8‑BOM编码的CSV文件。

## 方法说明

1. **数据清洗**：读取原始Excel文件，统一列名格式，检查并处理缺失值（无缺失），确保“指标”列为字符串并去除首尾空格，保存清洗后数据至指定路径。
2. **描述性统计**：计算各月份指数的均值、标准差、中位数等。
3. **假设检验**：鉴于每组样本量较小（每月仅13个观测），全部采用非参数检验：
   - **Kruskal‑Wallis H检验**：检验5个月份的指数分布是否存在差异。检验基于65个观测值（总N）分为5组（每月一组，每组n=13）。
   - **Mann‑Whitney U检验**：若Kruskal‑Wallis显著，则进行两两比较，并使用Bonferroni法校正多重比较（α = 0.05 / 10 = 0.005）。
   - **效应量**：计算Kruskal‑Wallis的epsilon‑squared（ε²），近似于η²。
   - **中位数差异估计**：计算Hodges‑Lehmann中位数差异（基于所有配对差值的中位数）及其Bootstrap 95%置信区间（以10月与9月为例）。
4. **可视化**：
   - **折线图**：展示各指标在5个月份的趋势变化（前5项指标添加图例以避免重叠）。
   - **箱线图**：呈现各月份指数的分布情况。
   - **热力图**：以颜色深浅直观显示各指标在不同月份的具体数值。
5. **统计软件**：Python 3.11，使用pandas、numpy、scipy.stats、matplotlib、seaborn库。

## 统计学治理说明

- **样本量澄清**：总观测数N=65（13指标×5月），分为5组（每月），每组n=13。Kruskal‑Wallis检验比较5组间的分布。
- **小样本警告**：每组仅13个观测，属于小样本，因此严格采用非参数检验，避免对正态分布做出未经检验的假设。
- **多重比较校正**：若进行两两比较，采用Bonferroni校正，控制族错误率（FWER）。
- **效应量与置信区间**：报告假设检验时，同时提供检验统计量、p值、效应量及95%置信区间，避免仅依赖p值决策。
- **因果语言限制**：本分析为观察性研究，未进行随机实验或因果识别，因此结论仅描述关联性，不使用“导致”“影响”等因果性表述。

## 核心假设检验结论

1. **Kruskal‑Wallis检验**：
   - H统计量 = 1.1161，p = 0.8917，ε² = 0.0174。
   - **结论**：在5个月份间，13项消费价格指数的分布无统计学显著差异（p > 0.05）。由于p值远大于0.05，效应量估计不可靠，因此不强调效应量大小。
2. **两两比较（Bonferroni校正）**：
   - 所有10对月份比较的Mann‑Whitney U检验p值经Bonferroni校正后均为1.0，远大于校正后α=0.005。
   - **结论**：任意两个月之间的指数分布均无显著差异。
3. **中位数差异估计（2025年10月 vs 9月）**：
   - Hodges‑Lehmann中位数差异 = 0.5000（即10月比9月的中位数高约0.50单位）。
   - Bootstrap 95% CI = [‑0.4500, 1.5000]。
   - **结论**：置信区间包含零，与假设检验一致，支持两月间无显著中位数差异。

## 结果解释

- **整体趋势**：各月份指数的均值在100.73至101.43之间波动，中位数在100.10至100.90之间，表明各月整体价格水平相对稳定。
- **月份差异**：统计检验未发现月份间的显著差异，提示2025年6月至10月期间，消费价格指数的同比变化在统计上保持平稳。
- **类别差异**：从热力图与折线图可见，部分类别（如“食品烟酒类”）指数较低（接近97‑100），而“交通和通信类”等类别指数较高（接近108‑112），显示不同消费类别的价格变化幅度存在明显差异。

## 讨论

- **政策含义**：在观察期内，整体消费价格指数未出现显著的月度波动，可能反映宏观经济价格环境相对稳定。但不同类别间的差异提示结构性价格变化值得关注，例如食品烟酒类价格同比增幅较低，而交通通信类增幅较高。
- **方法局限**：
  - 每组样本量较小（每月仅13个指标），统计检验力有限，可能无法检测到细微的月份差异。
  - 数据仅覆盖5个月，难以推断长期趋势或季节性模式。
  - 指标为同比指数，已消除季节性影响，但无法反映环比变化。
- **后续建议**：
  - 若可获得更长时间序列（如24个月以上），可进行时间序列分析（如ARIMA）或季节性分解。
  - 可进一步分析各类别指数与宏观经济变量（如货币供应、就业率）的相关性。
  - 考虑使用贝叶斯方法在小样本下提供更稳健的区间估计。
