# 居民消费价格指数 (CPI) 月度数据分析报告

## 数据概览

- **数据来源**：中国居民消费价格指数 (CPI) 月度数据，包含13个分类指标，时间跨度为2025年6月至2025年10月（共5个月）。
- **数据规模**：13行（指标）× 6列（1列指标名称 + 5个月份）。
- **数据格式**：每个指标值为“上年同月=100”的指数，反映同比变化。
- **样本量警告**：样本量极小（N=13），所有统计推断需高度谨慎，优先采用非参数方法。

## 方法说明

1. **数据清洗**：读取原始Excel文件，检查缺失值与重复行，按时间顺序重排月份列，保存为CSV格式。
2. **描述性统计**：计算各月平均值、标准差、极差，以及各指标跨月均值与标准差。
3. **假设检验**：
   - 采用 **Friedman 检验**（非参数重复测量方差分析）检验5个月份间是否存在显著差异。
   - 若Friedman检验显著，进行 **Wilcoxon 符号秩检验** 作为事后两两比较，并应用 **Bonferroni 校正** 控制多重比较错误。
4. **效应量**：计算 **Kendall's W** 作为Friedman检验的效应量，衡量月份间差异的一致性程度。
5. **可视化**：
   - 热图展示各指标在各月的数值分布。
   - 折线图展示各月平均值的变化趋势（带误差棒）。
   - 横向条形图展示平均值最高的前10个指标。

## 统计学治理说明

- **小样本处理**：因N=13 < 30，全程采用非参数检验，避免正态分布假设。
- **多重比较校正**：进行10对两两比较，使用Bonferroni校正（校正后α = 0.005）。
- **效应量与置信区间**：报告Kendall's W效应量，并注明因样本量过小未计算其置信区间，解释时需谨慎。
- **因果声明限制**：本分析仅描述相关性，不推断因果关系。

## 核心假设检验结论

### Friedman 检验结果
- **检验统计量**：χ² = 8.6154
- **p值**：p = 0.0716
- **效应量（Kendall's W）**：W = 0.1657
- **95% CI**：因样本量过小未计算，点估计解释需谨慎。

**结论**：在α=0.05水平上，Friedman检验不显著（p = 0.0716 > 0.05），未能拒绝零假设。即**没有足够证据表明13个CPI指标在2025年6月至10月这5个月份之间存在整体差异**。

### 事后两两比较
由于整体检验不显著，未进行正式的事后两两比较。若进行Wilcoxon符号秩检验并应用Bonferroni校正（校正后α=0.005），所有月份对均未达到校正后的显著性水平。

## 结果解释

1. **描述性统计**：
   - 各月平均CPI在100.73至101.43之间波动，10月平均值最高（101.43），6月最低（100.73）。
   - 各月标准差逐渐增大（2.54至3.59），表明10月各指标分化程度最高。
   - 跨月平均值最高的指标为“其他用品及服务类居民消费价格指数”（均值约108.5），最低为“食品烟酒类居民消费价格指数”（均值约98.5）。

2. **假设检验解释**：
   - Friedman检验p值=0.0716，虽未达到0.05显著性阈值，但接近临界值，提示可能存在微弱的时间趋势。
   - 效应量Kendall's W=0.1657，属于小到中等效应，表明月份间的差异一致性较低。
   - 鉴于样本量极小（仅13个指标），统计检验力不足，即使存在真实差异也可能无法检测到。

3. **可视化洞察**：
   - 热图显示多数指标在100附近，但“其他用品及服务类”“教育文化娱乐类”等指标持续高于105，而“食品烟酒类”持续低于100。
   - 月度趋势线显示平均值从6月到10月缓慢上升，但误差棒重叠，变化不显著。
   - 条形图清晰展示了高CPI指标（如其他用品及服务、教育文化娱乐）与低CPI指标（如食品烟酒、交通通信）的对比。

## 讨论

1. **方法学局限**：
   - 样本量极小（N=13），所有统计推断的稳定性与泛化能力受限。
   - 仅5个月份，时间序列过短，难以捕捉季节性或多期趋势。
   - 数据为同比指数（上年同月=100），反映的是同比变化，而非环比或绝对水平。

2. **实际意义**：
   - 各CPI分类指标表现分化：服务类、教育文化娱乐类价格同比上涨明显，而食品烟酒类同比下跌，反映消费结构变化与政策影响。
   - 月份间整体差异不显著，说明在2025年6‑10月这五个月内，CPI整体同比变化相对平稳，未出现剧烈波动。
   - 标准差逐月扩大可能提示部分指标（如其他用品及服务）波动加剧，值得后续关注。

3. **因果提醒**：
   - 本分析仅描述各CPI指标与月份的相关性，**不能推断月份导致CPI变化**。
   - 价格变化受多重因素（货币政策、供需关系、季节性、政策调控等）影响，需更长期、更细粒度的数据与因果推断设计才能探究驱动因素。
